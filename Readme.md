# Crack Chad's WiFi password

Compile with:

```
javac Main.java -classpath core-3.3.1.jar
```

Run with:

```
java -classpath core-3.3.1.jar:. Main qr.rgb
```

"core-3.3.1.jar" is the barcode decoder from [zxing](https://github.com/zxing/zxing).
